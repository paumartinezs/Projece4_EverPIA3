
# Introducció

Benvinguts, equip.

Durant les darreres setmanes, hem implementat serveis complexos per als nostres clients: servidors de fitxers (**NFS**), impressió (**CUPS**), autenticació (**LDAP**), etc. Però cap d'aquests serveis avançats no pot funcionar de manera fiable si **els fonaments del sistema operatiu són febles**.

Un servidor amb:
- una configuració de xarxa incorrecta,  
- permisos mal assignats, o  
- paquets sense actualitzar  

és un **risc de seguretat** i una **futura incidència garantida**.  
A EverPia, no podem lliurar una feina així; **la nostra reputació es basa en la robustesa.**

---

# La Vostra Missió: Auditoria i Estandardització del Servidor Base

Abans de desplegar el proper servei avançat, farem un pas enrere per **consolidar la base**. Aquesta activitat és una auditoria interna i una pràctica del nostre **"Protocol de Desplegament Estàndard" (SOP – Standard Operating Procedure).**

Heu de demostrar que sou capaços de:
- desplegar una nova màquina virtual (Ubuntu Server) des de zero, i  
- aplicar el nostre **protocol de configuració bàsica**.

Aquestes són les tasques fonamentals que tot consultor júnior ha de dominar abans de tocar l'entorn d'un client. Les repassarem per assegurar-nos que tots estem al mateix nivell d'excel·lència.

**Prepareu-vos:** un cop finalitzada la fase d’entrenament haureu de superar una prova pràctica de validació per obtenir la vostra certificació **SOP-Specialist**.
